Fallback installer

Ubuntu 22.04 desktop one-liner for lazy admins.

Use this when the custom ISO path is taking too long. Install a clean Ubuntu 22.04 Desktop machine, get it on Wi-Fi or Ethernet, open this page, and run one command to lay down the THRIL kiosk packages and setup.

Open hosted script Back to main docs

Run this

Recommended command: 

wget -qO- https://techmore.github.io/thril/scripts/ubuntu-22.04-desktop.sh | sudo bash

Alternate command

If `wget` is missing or you prefer `curl`:

curl -fsSL https://techmore.github.io/thril/scripts/ubuntu-22.04-desktop.sh | sudo bash

What it does

  • Installs `cups` and `cockpit` on a stock Ubuntu 22.04 Desktop box.
  • Downloads the current THRIL setup payload from the website.
  • Configures autologin, Packit launcher, Chromium policy, and printer bootstrap services.
  • Installs Chromium through the same hosted setup logic used by the remote autoinstall flow.
  • Leaves Wi-Fi alone so it does not clobber an already-working desktop station.

Notes

  • This is intended for Ubuntu 22.04 Desktop, not server.
  • Run it only after the machine already has internet access.
  • The script logs to `/var/log/thrill-desktop-bootstrap.log`.
  • A reboot is recommended after the script finishes.
